# Pattern Printing in C

This repository contains a collection of Pattern Printing programs implemented in C. The patterns are categorized into **Linear Patterns** and **Non-Linear Patterns**, helping beginners strengthen their understanding of loops, nested loops, conditional statements, and logical thinking.

## 📌 Overview

Pattern printing is one of the most effective ways to master programming fundamentals. These programs demonstrate the use of:

* `for` loops
* Nested loops
* Conditional statements (`if-else`)
* Number patterns
* Alphabet patterns
* Symbol patterns
* Row and column manipulation

## 📂 Categories

### 1. Linear Patterns

Patterns where elements follow a straightforward sequence.

#### Examples

* Number patterns
* Alphabet patterns
* Sequential increment patterns
* Continuous number series
* Odd and even number patterns

Example:

1 2 3 4
5 6 7 8
9 1 2 3

### 2. Non-Linear Patterns

Patterns involving conditions, diagonals, symmetry, or complex arrangements.

#### Examples

* Star (`*`) patterns
* Pyramid patterns
* Inverted pyramids
* Diamond patterns
* Hollow patterns
* Diagonal patterns
* Mixed symbol patterns (`*`, `#`, `@`)
* Conditional row and column patterns

Example:

* * * #
* * # @
* # @ @

# @ @ @

## 🚀 How to Run

1. Clone the repository.
2. Open the source file.
3. Compile the program:

   ```
   gcc filename.c -o output
   ```
4. Run:

   ```
   ./output
   ```
